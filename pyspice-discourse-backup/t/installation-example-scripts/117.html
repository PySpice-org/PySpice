Dear PySpice community, I would like to share some installation example files to illustrate the installation procedure of PySpice on Linux by using apt and pip only.

Those files are published as Dockerfile scripts since they allow full automatic machine instantiation with docker.
They cover both Ubuntu and Debian, using a standard approach using only apt :

  • GitHub - cyber-g/debian-pyspice-std: A Dockerfile to demonstrate the installation of PySpice on Debian
  • GitHub - cyber-g/ubuntu-pyspice-std: A Dockerfile to demonstrate the installation of PySpice on Ubuntu

One can also find a more ‚Äúmanual‚Äù approach to demonstrate how to properly set the PYTHONPATH and LD_LIBRARY_PATH environment variables :

  • GitHub - cyber-g/debian-pyspice-semi-custom: A Dockerfile to demonstrate the installation of PySpice on Debian using a "manual" installation of libngspice
  • GitHub - cyber-g/ubuntu-pyspice-semi-custom: A Dockerfile to demonstrate the installation of PySpice on Ubuntu using a "manual" installation of libngspice

The two latter should be useful for those who want to install PySpice on Google Collab.

If you have docker on your computer, you can also pull directly the images from Docker Hub :

  • Docker Hub PySpice Debian
  • Docker Hub PySpice Ubuntu

Enjoy !
